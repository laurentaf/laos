# ADR-002: LinkedIn Strategy — McDonald's 3-Layer Guardrails + Format Wheel

**Date:** 2026-06-21  
**Status:** Accepted  
**Author:** Laurent.io (orchestrator)

## Context

Laurent proposed a "McDonald's 3-layer guardrails" framework for LinkedIn content. The question was whether to adopt it as-is or modify it.

Options considered:
- **Adopt as-is** — simple, clear, 3 content types
- **Reject and propose alternative** — build new from scratch
- **Adopt with modifications** — enhance with format, metrics, and brand pillars

## Decision

Adopt with modifications = **McDonald's v1.1**:

- Keep 3 layers (Build, Case Study, Thesis) as content philosophy
- Add **format wheel** (W1 visual, W2 narrative, W3 build, W4 interactive)
- Add **brand pillar** ("Healthcare AI Data Engineer")
- Add **monthly metrics** (engagement > 3%, recruiter DMs > 5/month)
- Add **Dunning-Kruger gate** (Layer 3 requires GitHub evidence)

## Alternatives considered

- "Post about everything" (too broad, no differentiation)
- "Only open-source contributions" (too narrow, no personal brand)
- "Curate others' content" (zero differentiation, algorithm penalty)

## Consequences

- ✅ Framework is testable — metrics will validate at 90 days
- ✅ Niche positioning (healthcare + AI DE) is defensible in BR market
- ⚠️ Requires consistent creation + portfolio building (high effort)
- ⚠️ Layer 3 may trigger debate → need thick skin for public takes

# Contract — README Portfolio Improvement

## Brief

**Project:** Meta-project to improve all public READMEs in the `laurentaf` GitHub portfolio.

**Brief:** An audit revealed 4 CRITICAL READMEs (laecon, ladesign, lan8n, laos-brand scoring ≤ 5/20) dragging down the portfolio average to 11.5/20. The LAOS README scores 20/20 and serves as the gold standard. This project rewrites and improves all 15 public READMEs in a tiered approach, raising the portfolio average to ≥ 14.5/20.

## Needs

**Needs:** design (visual quality of READMEs), documentation-lookup (industry best practices)

## Deliverables

**Deliverables:**
- 4 rewritten CRITICAL READMEs (laecon, ladesign, lan8n, laos-brand)
- 3 improved HIGH READMEs (laengine, pizzarias-sp-2026, hospital-viana-claims)
- 4 improved MEDIUM READMEs (logistica-me, latade, emanuella-stock-ingestion, semana-ai-data-engineer)
- 3 polished LOW READMEs (giovanna-rupture-monitor, abandono-academico, laurentaf profile)
- 1 reusable README template (`knowledge/readme-templates.md`)

## Capabilities

**Capabilities used:** LADESIGN (dashboard-designer), LATADE (data-architect for data-heavy repos)

## Repo

**Repo:** This is a meta-project — artifacts live in `projects/_meta/readme-improvement/`. Actual README changes are pushed to each repository via `github` MCP.

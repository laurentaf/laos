# ADR Template

Copy this file to `NNN-<slug>.md` (e.g., `003-my-decision.md`) and fill in.

## Template Fields

| Field | Description |
|-------|-------------|
| Status | proposed / accepted / rejected / deprecated |
| Date | YYYY-MM-DD |
| Decisor | Who decided (LACOUNCIL, author, etc.) |
| Proposal | LACOUNCIL proposal ID if applicable |

## Required Sections

### Contexto
What prompted the decision? Background, problem statement, constraints.

### Decisão
What was decided? Be specific — include naming, versioning, technology choices.

### Alternativas Consideradas
At least 3 alternatives with why each was rejected.

### Consequências
- **Positivas:** what improves
- **Custos e responsabilidades:** what gets harder / new maintenance burden
- **Riscos:** what could go wrong and mitigations
